U.S. Flag Code

Medium: Website

Date: January 2025

This project reimagines the U.S. Flag Code as a readable, interactive site. The original document is dense and legalistic, but the flag itself is everywhere—on buildings, clothing, packaging. I wanted to bridge that gap and create an experience that feels both accessible and visually rooted in American design history. The site uses a simple one-column layout and vintage-inspired typography to bring clarity and tone. The goal was to take the flag code out of its legal shell and into cultural context through layout, type choices, and an emphasis on readability. This project sits at the intersection of civic education, design history, and national symbolism.
